# Web-OBD or OBD Dataview
Web-OBD or OBD Dataview is a system developed to display data from a vehicle's OBD system via a web interface. It connects to an OBD device or OBD data source, processes the data, and displays it through a web interface.
Designed to monitor data from a vehicle's On-Board Diagnostics (OBD), such as engine RPM, speed, temperature, trouble codes, etc., in real-time via a web interface, or to store recorded data.
